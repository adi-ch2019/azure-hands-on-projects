# AI Cloud Governance & Orchestration with AI

## Azure Automated VM Provisioning and Governance with AI

### Implements : This project demonstrates how to automate the provisioning and governance of Azure Virtual Machines (VMs) using Azure Runbooks and Event Grid

This project showcases how Azure's automation tools can be used to streamline the management of Azure VMs, ensuring that they are provisioned and governed effectively while **minimizing manual intervention**.
